# RetailPulse Final Report

## Project Overview
RetailPulse analyzes retail customer behavior, predicts churn, segments customers, and recommends inventory reorder quantities.

## Modules Completed
- EDA
- Feature Engineering
- Customer Segmentation
- Demand Forecasting
- Churn Prediction
- Inventory Optimization
- Streamlit Dashboard

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Plotly, Streamlit

## Conclusion
RetailPulse helps businesses understand customers, reduce churn, and improve inventory planning.
